## Anti-Patterns Verdict
Feel: not slop (soft palette, calm IA, sophisticated PermissionBuilder). Details: tells slipped in.
- LLM: tiny uppercase tracked eyebrow on all 7 dashboard cards (style.css .card .label outspecifies plain .label); identical equal-weight card grid; metrics duplicated to fill grid; gradient brand dot.
- Detector (detect.mjs over ui/src): 1 finding — side-tab (side-stripe border) on components/Toasts.svelte. Corroborates the LLM. No false positives.
- No browser overlay injected; inspected via a11y snapshots + console + CLI detector; CSS-level contrast/responsive analysis is the authoritative visual source.

## Priority Issues
- [P0] Accessibility floor fails AA (goal was AAA): --text-faint ~2.3-2.6:1; dark-mode primary button text 2.91:1; semantic badges fail (success 2.73, warning 2.33, danger 2.8); PermissionBuilder bad-JSON warning fails AA; focus-visible invisible in both themes; .nav-link no focus style; zero prefers-reduced-motion. -> /colorize + /harden + /audit
- [P0] Destructive/secret actions have no confirmation or gravity: bucket delete one-click; Deactivate/Rotate/Remove-policy instant; share link one-click; one-time secrets scroll off. Inverts Principle 2. -> /harden + /clarify
- [P0/P1] S3 jargon undercuts approachability: Logical/Physical bytes; raw bucket-policy JSON textarea; Advanced = raw s3: verbs; "Bearer token" unexplained at credential moment; em dashes/env jargon in copy. -> /clarify
- [P1] Flat hierarchy / wall of options: 7 undifferentiated dashboard cards (2 duplicate the panel); bucket Settings = 6 identical stacked panels (cognitive load HIGH). -> /layout + /typeset
- [P1] No loading/empty/error discipline: no skeletons; refresh flashes page; empty states don't teach; single shared error banner not announced; inputs have no invalid state. -> /polish + /harden + /onboard
- [P1] Looks-done-but-isn't + slop tell: object pagination display-only (cannot advance); dead var(--radius)/var(--accent); Monitoring->Replication mismatch; side-stripe toast border (detector-confirmed). -> /polish + /audit
